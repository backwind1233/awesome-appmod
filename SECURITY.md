# Security Policy

## Supported Versions

This project is currently in active development. Security updates will be applied to the latest version.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it to us privately:

1. **Do not** open a public issue
2. Email the details to the repository owner
3. Include steps to reproduce if possible

We will respond as quickly as possible and work with you to address the issue.

## Security Best Practices

When contributing tasks to this repository:

- Do not include secrets, API keys, or credentials
- Use environment variables for sensitive configuration
- Follow Microsoft's security guidelines
- Keep dependencies up to date

For more information, see [Microsoft Security Response Center](https://www.microsoft.com/msrc).
